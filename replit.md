# Nurney Plant and Civil Ltd Website

## Overview
A modern, responsive single-page website for Nurney Plant and Civil Ltd, a plant machinery hire and civil engineering company based in Nurney, Co Carlow, Ireland.

## Recent Changes
- **Feb 2026**: Added Featured Projects section with 3 project sub-pages
  - Radharc Ballyfin Road, Portlaoise
  - Sea View Court, Rocksborough, Wexford
  - St. Canices School, Kilkenny
- **Feb 2026**: Complete redesign using HTML5 and Tailwind CSS
  - Modern industrial aesthetic with orange/grey color palette
  - Single-page layout with smooth scrolling navigation
  - Masonry gallery for fleet/project images
  - Added Careers section with resume submission
  - Sticky mobile contact buttons
  - SEO-optimized meta tags

## Project Structure
- `index.html` - Main single-page website with all sections
- `project-ballyfin.html` - Radharc Ballyfin Road project page
- `project-seaview.html` - Sea View Court project page
- `project-st-canices.html` - St. Canices School project page
- `PrivacyPolicy.html` - Privacy policy page
- `sitemap.xml` - SEO sitemap
- `assets/images/` - Company images, fleet photos, project images
- `new_assets/` - Project-specific photos for featured projects

## Sections (Home Page)
1. **Hero** - High-impact headline with CTA buttons
2. **Services** - Plant Machinery Hire and Civil Engineering cards
3. **Featured Projects** - 3 project cards linking to sub-pages
4. **Gallery** - Masonry grid of fleet and project images
5. **About** - Company info and trust signals
6. **Careers** - "We Are Hiring" section with mailto link
7. **Contact** - Phone, email, location details
8. **Footer** - Copyright, privacy policy, Facebook link

## Tech Stack
- HTML5 with semantic tags
- Tailwind CSS (via CDN)
- Vanilla JavaScript for mobile menu
- Static hosting
